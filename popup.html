사용 방법

1. 지도 서비스 접속
아래 버튼으로 네이버지도, 카카오맵 중 하나를 선택하세요

2. 장소 선택
지도에서 저장하고 싶은 장소를 클릭하여 상세 정보를 확인하세요

3. 목록 관리
우측 사이드바에서 '새 목록 생성' 버튼으로 목록을 만들거나 기존 목록을 선택하세요

4. 장소 저장
사이드바의 '현재 보고 있는 장소 추가' 버튼을 클릭하면 장소명, 카테고리, 별점이 자동으로 저장됩니다

5. 💎 커스텀 필드 관리 (핵심 기능)
이 앱의 가장 강력한 기능입니다! '커스텀 필드 관리'에서 나만의 분류 체계를 만들어보세요:
• 텍스트형 필드: 가격대, 주차정보, 특이사항 등 자유로운 정보 입력
• 선택형 필드: 방문여부(가봤음/안가봤음), 평점(1-5점), 우선순위(높음/보통/낮음) 등
• 맞춤형 데이터베이스: 맛집 탐방, 여행 계획, 부동산 검색 등 용도에 맞게 완전히 커스터마이즈 가능

6. 추가 정보 입력
저장된 장소에 상세한 메모를 추가하고, 설정한 커스텀 필드에 값을 입력하세요

7. 데이터 백업
'데이터 내보내기'로 CSV나 JSON 형식으로 백업하고, 다른 기기에서 '데이터 가져오기'로 복원할 수 있습니다
