# CustomPlaceDB 🗺️

커스텀 필드로 나만의 장소 데이터베이스를 만들어보세요

![Version](https://img.shields.io/badge/Version-1.0-brightgreen?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-blue?style=flat-square)

## 🌟 주요 기능

### 💎 커스텀 필드 시스템 (핵심 기능)
- **텍스트형 필드**: 가격대, 주차정보, 특이사항 등 자유로운 정보 입력
- **선택형 필드**: 방문여부, 평점, 우선순위 등 미리 정의된 옵션 선택
- **맞춤형 데이터베이스**: 맛집 탐방, 여행 계획, 부동산 검색 등 용도별 완전 커스터마이즈

### 🗺️ 다중 플랫폼 지원
- **네이버지도** 🟢 - 가장 안정적인 데이터 추출
- **카카오맵** 🟡 - 실시간 DOM 파싱
- **구글지도** 🔵 - 국제적 장소 정보 지원

### 📊 데이터 관리
- **자동 정보 추출**: 장소명, 카테고리, 별점 자동 수집
- **목록별 관리**: 여러 목록으로 장소 분류
- **메모 기능**: 각 장소에 개별 노트 추가
- **백업/복원**: CSV, JSON 형식으로 데이터 내보내기/가져오기

## 🚀 설치 방법

### Chrome 웹스토어에서 설치 (추천)
*Chrome 웹스토어 등록 준비 중*

### 개발자 모드로 설치
1. 이 저장소를 다운로드하거나 클론
2. Chrome에서 `chrome://extensions/` 접속
3. 우상단 **개발자 모드** 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. CustomPlaceDB 폴더 선택

## 📖 사용 방법

### 1️⃣ 지도 서비스 접속
확장프로그램 팝업에서 네이버지도, 카카오맵, 구글지도 중 선택

### 2️⃣ 장소 선택
지도에서 저장하고 싶은 장소를 클릭하여 상세 정보 확인

### 3️⃣ 목록 관리
우측 사이드바에서 '새 목록 생성' 또는 기존 목록 선택

### 4️⃣ 장소 저장
사이드바의 '현재 보고 있는 장소 추가' 버튼 클릭
→ 장소명, 카테고리, 별점이 자동으로 저장됩니다

### 5️⃣ 💎 커스텀 필드 활용 (핵심!)
- **커스텀 필드 관리**에서 나만의 분류 체계 생성
- 텍스트형/선택형 필드로 상세 정보 입력
- 맛집, 여행지, 부동산 등 용도별 맞춤 설정

### 6️⃣ 데이터 백업
'데이터 내보내기'로 CSV/JSON 백업 및 다른 기기에서 복원

## 🎯 활용 사례

### 🍽️ 맛집 탐방
```
목록: "가고 싶은 맛집", "다녀온 맛집"
커스텀 필드: 
- 방문여부 (가봤음/안가봤음)
- 나의 평점 (1-5점)
- 가격대 (저렴/보통/비쌈)
- 주차 가능 여부
```

### ✈️ 여행 계획
```
목록: "서울 여행", "부산 여행"
커스텀 필드:
- 우선순위 (높음/보통/낮음)
- 동행자
- 예상 비용
- 방문 계획일
```

### 🏠 부동산 탐색
```
목록: "아파트 후보", "오피스텔 후보"
커스텀 필드:
- 가격대
- 교통편 점수
- 장단점
- 연락처
```

## 🛠️ 기술 스택

- **Manifest V3** - 최신 Chrome Extension API
- **Vanilla JavaScript** - 순수 자바스크립트로 경량화
- **Chrome Storage API** - 로컬 데이터 저장
- **DOM Manipulation** - 실시간 데이터 추출
- **CSS3** - 반응형 UI 디자인
